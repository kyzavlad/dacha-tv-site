# Admin Panel Content Guide

Цей документ пояснює, в якому порядку та як вносити контент в адмін-панель сайту Дача TV. Дотримуйтесь порядку — деякі розділи залежать від інших.

---

## Доступ до адмін-панелі

Відкрийте адмін-панель за адресою:
```
https://ваш-домен.vercel.app/admin
```
або локально — `http://localhost:3000/admin`.

Введіть пароль з `ADMIN_PASSWORD` (змінна середовища у Vercel).

---

## Крок 1 — Налаштування сайту

**Де:** Admin → Налаштування

Заповніть в першу чергу:

| Поле | Що вводити | Приклад |
|------|-----------|---------|
| Номер телефону | Лише ваш реальний телефон | `+380671234567` |
| Адреса (повна) | Для контактів та підвалу | `Коротич, Пісочинська ОТГ, Харківська обл.` |
| Telegram | Посилання на канал або особистий | `https://t.me/dachatv` |
| YouTube | Головна сторінка каналу | `https://youtube.com/@dachatv` |

Після збереження телефон одразу з'явиться в шапці та підвалі сайту.

---

## Крок 2 — Мед

**Де:** Admin → Мед

Це найважливіший розділ. Сайт показує порожній каталог, поки немає жодного продукту.

### Обов'язкові поля

| Поле | Опис |
|------|------|
| Назва | Повна назва. Наприклад: `Мед Акація` |
| Slug | Унікальний ідентифікатор URL (латиниця, дефіси). Наприклад: `akacia` |
| Сорт | Визначає текст і нотки на картці товару |
| В наявності | Вимкніть якщо сорту немає на складі |
| Порядок | Число: 1 для першого в списку |

### Рекомендований порядок додавання

1. Акація (порядок: 1)
2. Липа (порядок: 2)
3. Сонях (порядок: 3)
4. Різнотрав'я (порядок: 4)
5. Сади (порядок: 5)
6. Ліс (порядок: 6)

### Фото

- Завантажуйте фото через поле "Зображення" на сторінці редагування продукту
- Розмір: мінімум **800×800px**, квадрат
- Формат: JPG або WebP

### Показати на головній

Поле **"На головній"** (`is_featured`) — увімкніть для 3–4 сортів. Саме вони відображатимуться у блоці "Популярні сорти" на головній сторінці.

## Крок 3 — Відгуки

**Де:** Admin → Відгуки

### Як додавати

1. Ім'я покупця (лише ім'я або ім'я + перша літера прізвища)
2. Місто
3. Текст відгуку — реальні слова, без редагування (окрім видалення особистих даних)
4. Оцінка (1–5 зірок)
5. Поле **"Видимий"** — залиште вимкненим поки не перевірите відгук

> ⚠️ Відгуки не з'являться на сайті, поки ви не увімкнете "Видимий".

---

## Крок 4 — Продукти пасіки

**Де:** Admin → Продукти пасіки

Це пилок, прополіс, перга, горіхи в меду тощо. Заповнюйте після меду.

- Назва: `Пилок квітковий`, `Прополіс`, `Перга` тощо
- В наявності: вмикайте/вимикайте за потреби
- Порядок: число, менше = вище в списку

---

## Крок 5 — Для пасічників

**Де:** Admin → Для пасічників

Бджолопакети, бджолосім'ї, вулики. Якщо зараз не сезон — пропустіть, поверніться навесні.

Обов'язково вкажіть **Тип** (`package`, `colony`, `hive`) — він визначає, в який розділ потрапить товар на сторінці.

---

## Крок 6 — FAQ

**Де:** Admin → FAQ

Рекомендовані питання для старту:

**Категорія: products**
- Як зберігати мед?
- Чи натуральний ваш мед?
- Чому мед засахарився?

**Категорія: ordering**
- Як замовити мед?
- Яка мінімальна кількість для замовлення?

**Категорія: delivery**
- У які міста ви доставляєте?
- Яка вартість доставки?

Завжди заповнюйте поле **Порядок** — це визначає, яке питання буде вгорі в категорії.

## Що перевірити після внесення контенту

- [ ] Сайт показує ваш телефон у шапці та підвалі
- [ ] Сторінка `/honey` показує ваші продукти, а не порожній каталог
- [ ] Кнопка "Замовити" відкриває форму (перевірте в мобільному браузері)
- [ ] Відгуки з'являються на головній (якщо є видимі)
- [ ] `/faq` показує ваші питання і відповіді

## Часті питання

**Зміни не з'являються на сайті?**
Сторінки з продуктами кешуються. Зачекайте хвилину і оновіть. Або зробіть Redeploy у Vercel.

**Як видалити продукт?**
Admin → Мед → знайдіть продукт → кнопка "Видалити". Але краще просто вимкнути "В наявності" — так продукт зникне з каталогу але залишиться в базі.

**Як завантажити фото?**
На сторінці редагування продукту є поле для завантаження зображення. Фото зберігаються у Supabase Storage (bucket `site-media`).
